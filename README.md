# FriendFinder
Friend Finder is like the Tinder for finding friends! The user takes a short 10-question survey and answers questions based on a 1-5 scale, where 1 is Stongly Disagree and 5 is Strongly Agree. After the survey is complete, their closest friend match will pop up!
